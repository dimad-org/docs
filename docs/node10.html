Translation to COSY input

The keyword COSY can be used after a beamline has been selected in the MAD input stream with the call

use, beamline name

and before the keyword dimat or newbeamline.

The keyword COSY will prompt the program to provide an input stream for the chosen beamline in a format compatible with the COSY program [7]

Not all elements are currently translated. When an element is not translated it is replaced by a drift.

The same keyword can also be used in the dimad operation input stream See the appropriate listing in the following section.
