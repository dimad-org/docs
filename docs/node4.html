General Syntax

When describing the machine, a statement can be continued on the next line by ending the current line with a "&".

A comment line begins with one of the characters "!", "*","(","@" in the first column.

A ";" is used to separate statements on the same line. Keywords are uniquely specified by the first five letters, and only those five must be entered. At most 8 letters in a keyword are checked.

The keyword NOECHO can be used to suppress transmission of the input data stream to the output files. The keyword ECHO reinstates the stream of the input data to the output files.
