Element Definitions

To define an element, 
        label: type [,pkeyw=value,......]
where label is the name of the element, type is an element type (see below) and pkeyw is a parameter keyword appropriate for the element type (see below). value can be a parameter name, or an expression involving parameters. 

Examples:


b: sbend, l=lb, angle=lb/rho 


d0: drift
